The topic of the project is File Management system.

You must write the code under Linux environment and use C language to implement the file system.

The project must include a written portion that describes a file management system, example of the modern file management systems and description of the file management system that you are implementing.

You must be able to create files, open files, close files, search for files.

You must know how to divide the task within your group members.

At the end:

You must have a formal report: Abstract, introduction, body and conclusion.  In the body, you can talk about the code and results.

You must have a working program that shows the implementation of a file system.

You must research on other files systems that we have and compare them. What is the file system used in MAC and the file systems used in PCs and the file system that your flash drive has.

You must talk about what does happen when you copy a file from one file system into another file system. Is there any information lost during this process?

Conclusion: What you have learned in terms of the research work, the group work and summarize the results and findings.

Project presentation: You must show to me and other class members how your project work and talk about your project -- project report will be due at the time that you do the presentation,
